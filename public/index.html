🚨 Incident Response Playbook
แนวทางปฏิบัติเมื่อเกิดเหตุ — ตาม MOPH CIRP Framework
🔒 Ransomware Readiness Score
คะแนน: — —
⚠ ระยะที่ 1 — ตรวจพบ & ยืนยัน
□ แจ้งเตือนจาก Wazuh/AV/EDR ปรากฏขึ้น
□ ตรวจสอบ extension ไฟล์ที่เปลี่ยนแปลง
□ หา ransom note (README.txt, !DECRYPT.txt)
□ ยืนยัน process แปลกที่ CPU สูง
□ บันทึกเวลา T0 (Time of Detection)
🔌 ระยะที่ 2 — Containment
□ ตัดเครือข่ายทันที — unplug LAN / disable WiFi
□ Isolate VLAN ที่ได้รับผลกระทบ (FortiGate)
□ Disable user account ที่ถูก compromise
□ อย่า shutdown — เก็บ forensic memory ก่อน
□ แจ้ง ICT MOPH ภายใน 1 ชั่วโมง
🔍 ระยะที่ 3 — Investigation
□ เก็บ Wazuh logs ช่วง T0-2h ก่อนหน้า
□ ตรวจ Wazuh Rule 657 (firewall-drop) alerts
□ หา Initial Access: phishing/RDP/web exploit
□ ระบุ Patient Zero (เครื่องแรกที่ติด)
□ ส่ง sample ไฟล์ให้ NCSA / ThaiCERT
♻ ระยะที่ 4 — Recovery
□ Restore จาก Backup (3-2-1) offline copy
□ ตรวจสอบ Backup ไม่ติด ransomware ก่อน restore
□ Reset credentials ทุก account
□ Patch ช่องโหว่ที่ถูก exploit
□ Monitor เข้มข้น 72h หลัง restore
□ ทดสอบระบบก่อน reconnect เครือข่าย
📋 ระยะที่ 5 — Reporting
□ รายงาน ICT MOPH ภายใน 72h (PDPA requirement)
□ รายงาน NCSA / ThaiCERT ถ้ากระทบ Critical infra
□ จัดทำ After Action Report (AAR)
□ Update IRP/DRP จากบทเรียน
□ ซ้อมแผนรับมือครั้งถัดไป

async function syncBlacklist() {
  const NCSA = 'http://NCSA-HOST'
  const CF_ACCOUNT = 'YOUR_ACCOUNT_ID'
  const CF_LIST_ID = 'YOUR_LIST_ID'
  const CF_TOKEN  = 'YOUR_API_TOKEN'

  // Fetch NCSA IPs
  const r = await fetch(`${NCSA}/export/iptables`)
  const txt = await r.text()
  const ips = txt.split('\n')
    .filter(l => l.startsWith('-A INPUT'))
    .map(l => l.match(/-s\s+(\S+)/)?.[1])
    .filter(Boolean)
    .map(ip => ({ ip }))

  // Replace Cloudflare IP List
  await fetch(
    `https://api.cloudflare.com/client/v4/accounts/${CF_ACCOUNT}/rules/lists/${CF_LIST_ID}/items`,
    {
      method: 'PUT',
      headers: {
        'Authorization': `Bearer ${CF_TOKEN}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(ips.slice(0, 10000))
    }
  )
}

#!/bin/bash  # ncsa_tail_watch.sh
# Real-time web log scanning against NCSA blacklist
NCSA="http://NCSA-HOST"
LOGFILE="/var/log/nginx/access.log"
BATCH=50  # check every 50 unique IPs

declare -A CHECKED

tail -F "$LOGFILE" | while read line; do
  IP=$(echo "$line" | grep -oE '^[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+')
  [ -z "$IP" ] && continue
  [ "${CHECKED[$IP]}" ] && continue
  CHECKED[$IP]=1

  RESULT=$(curl -sf --max-time 3 "$NCSA/check/auto/$IP")
  BLACKLISTED=$(echo $RESULT|grep -c '"blacklisted":true')

  if [ "$BLACKLISTED" -gt 0 ]; then
    echo "🔴 BLACKLISTED: $IP — $line"
    logger -t ncsa-watch "BLACKLISTED: $IP"
    # Optional: auto-block
    # iptables -I INPUT -s $IP -j DROP
  fi
done

import requests

NCSA = "http://NCSA-HOST"

def check_ip(ip):
    r = requests.get(
        f"{NCSA}/check/auto/{ip}",
        timeout=5
    )
    data = r.json()
    return data["blacklisted"], data.get("feeds",[])

def bulk_check(ips: list):
    r = requests.post(
        f"{NCSA}/check/bulk",
        json={"type":"auto","values":ips}
    )
    return {
        x["value"]: x["blacklisted"]
        for x in r.json()["results"]
    }

# Example:
blacklisted, feeds = check_ip("1.2.3.4")
hits = bulk_check(["1.2.3.4","9.9.9.9"])

function Bulk-Check($ips) {
    $body = @{
      type="auto";values=$ips
    } | ConvertTo-Json
    $r = Invoke-RestMethod `
      -Method POST `
      -Uri "$NCSA/check/bulk" `
      -Body $body `
      -ContentType "application/json"

}
